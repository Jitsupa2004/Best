# Network

## Definition

### English Definition 1

"A computer network is a group of interconnected computers and devices that can communicate and share resources with each other." — Britannica

### English Definition 2

"A computer network is a set of computers that are connected so they can share information and resources." — IBM

### Thai Definition

เครือข่ายคอมพิวเตอร์ (Network) คือ การเชื่อมต่อคอมพิวเตอร์และอุปกรณ์อิเล็กทรอนิกส์ตั้งแต่สองเครื่องขึ้นไปเข้าด้วยกัน เพื่อให้สามารถแลกเปลี่ยนข้อมูล ติดต่อสื่อสาร และใช้ทรัพยากรร่วมกันได้อย่างมีประสิทธิภาพ

## Explanation

Network หรือเครือข่ายคอมพิวเตอร์ เป็นระบบที่ช่วยให้อุปกรณ์ต่าง ๆ เช่น คอมพิวเตอร์ โทรศัพท์มือถือ เครื่องพิมพ์ และเซิร์ฟเวอร์ สามารถเชื่อมต่อและสื่อสารกันได้ โดยอาศัยสื่อกลางในการส่งข้อมูล เช่น สายเคเบิล ใยแก้วนำแสง หรือระบบเครือข่ายไร้สาย (Wi-Fi)

เครือข่ายคอมพิวเตอร์มีหลายประเภท เช่น LAN (Local Area Network) ซึ่งใช้ภายในบ้าน โรงเรียน หรือองค์กร และ WAN (Wide Area Network) ซึ่งครอบคลุมพื้นที่ขนาดใหญ่ โดยตัวอย่างที่สำคัญที่สุดคืออินเทอร์เน็ตที่เชื่อมโยงผู้คนและอุปกรณ์ทั่วโลกเข้าด้วยกัน

การใช้งาน Network ช่วยให้สามารถแบ่งปันทรัพยากร เช่น เครื่องพิมพ์ พื้นที่จัดเก็บข้อมูล และการเชื่อมต่ออินเทอร์เน็ตร่วมกันได้ นอกจากนี้ยังช่วยเพิ่มประสิทธิภาพในการสื่อสาร การทำงานร่วมกัน และการแลกเปลี่ยนข้อมูลในองค์กร ทำให้เครือข่ายคอมพิวเตอร์เป็นพื้นฐานสำคัญของเทคโนโลยีสารสนเทศในปัจจุบัน
